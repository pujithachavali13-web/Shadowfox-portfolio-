# Shadowfox-portfolio-

## Objective
To design and build a responsive personal portfolio website showcasing professional skills, current projects, and core technical backgrounds.

## Features
* **Brief Introduction:** A clean hero section introducing my engineering background.
* **Skills Showcase:** Visual cards displaying proficient programming tracks.
* **Portfolio Projects:** Grid layout highlights linking to current software work.
* **Resume Section:** Easy access link to view or download a professional CV.
* **Contact Information:** A clean input form designed for dynamic engagement.
